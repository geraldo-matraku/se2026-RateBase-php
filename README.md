## Teknologjitë e Përdorura

* PHP
* MySQL
* PHPMailer
* PHP Dotenv
* REST API

---

# Startimi i Backend-it

```bash
composer install
```

Për startimin e backend-it përdor XAMPP:

* Hap XAMPP Control Panel
* Start Apache dhe MySQL
* Vendos projektin backend në folderin `htdocs`
* Hape projektin në browser:

```bash
http://localhost/htdocs/sistem-vleresimi-produktesh-php/api/
```
